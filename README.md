# Multi-Step Form with React Context API

A modern, responsive, and state-managed multi-step form built with **React**, **Tailwind CSS**, and **Context API**. This project demonstrates how to handle complex form states across multiple components without "prop drilling."

## 🚀 Features
- **Centralized State Management:** Uses React Context API to share data between steps.
- **Modern UI:** Styled with Tailwind CSS for a clean and professional look.
- **Form Validation:** Integrated HTML5 validation for all inputs.
- **Responsive Design:** Optimized for both mobile and desktop screens.
- **Modular Components:** Clean and reusable component architecture.

## 🛠️ Tech Stack
- **Frontend:** React.js
- **State Management:** Context API
- **Styling:** Tailwind CSS

## 📂 Project Structure
- `context/`: Contains `FormContext` and `StepContext`.
- `components/`: Individual form steps (`PersonalInfo`, `EducationInfo`, `ProfessionalInfo`, `FormSummary`).
